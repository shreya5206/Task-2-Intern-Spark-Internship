# Algorithm Analysis Project

## Overview

This project demonstrates the implementation and analysis of fundamental algorithms in the areas of sorting, searching, and dynamic programming. The objective is to compare algorithm performance, understand time complexity, and evaluate runtime behavior across different input sizes.

## Implemented Algorithms

### 1. Merge Sort (Sorting)

* Uses the Divide and Conquer approach.
* Efficiently sorts large datasets.
* Time Complexity: **O(n log n)**

### 2. Binary Search (Searching)

* Searches for an element in a sorted array.
* Significantly faster than linear search for large datasets.
* Time Complexity: **O(log n)**

### 3. 0/1 Knapsack (Dynamic Programming)

* Solves an optimization problem by maximizing value within a given weight capacity.
* Demonstrates the use of dynamic programming techniques.
* Time Complexity: **O(n × W)**

## Features

* Clean and modular C++ implementations.
* Multiple test cases for each algorithm.
* Runtime analysis for varying input sizes.
* Complexity comparison and performance evaluation.
* Well-documented source code and report.
